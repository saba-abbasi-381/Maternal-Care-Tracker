# Smart Maternal Care Tracker

## Overview

Smart Maternal Care Tracker is a Python-based healthcare web application developed to support basic maternal care monitoring during pregnancy. This project helps estimate gestational age, trimester stage, expected delivery date (EDD), and provides pregnancy-related monitoring alerts and ultrasound reminders.

The goal of this project is to provide a simple, user-friendly, and informative system that can assist in pregnancy tracking and maternal care awareness.

---

## Features

### Gestational Age Calculation

Calculates pregnancy age in weeks and days using the Last Menstrual Period (LMP) and current date.

### Trimester Tracking

Automatically determines the pregnancy trimester:

* 1st Trimester
* 2nd Trimester
* 3rd Trimester
* Post-term Pregnancy

### Ultrasound Schedule Reminder

Provides reminders for important pregnancy scans, including:

* Confirmation ultrasound
* Fetal anomaly scan
* Fetal growth scan

### Monitoring & Alerts

Displays important pregnancy-related reminders such as:

* Vaccination reminders
* Monitoring recommendations
* Due-date alerts
* Maternal care guidance

### Expected Delivery Date (EDD)

Predicts the estimated delivery date using standard pregnancy duration calculations.

## How It Works

1. Enter the **Last Menstrual Period (LMP)** date.
2. Select the **Current Date**.
3. Click the **View button.**
4. The system displays:

   * Gestational age
   * Trimester
   * Scan recommendations
   * Monitoring alerts
   * Expected delivery date (EDD)
